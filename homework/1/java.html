Java is one of the most popular languages to learn.

It is an easy extension of C++, so it is an easy 2nd language to learn.

The language also has automatic memory management, which can't be said for languages like C and C++.

Java is executed in a virtual machine, making it platform independent.
All Java apps will run the same regardless of operating system.

Functions, called methods in Java, are called by foo(a) where foo is the name of a method and a is the argument being passed.
